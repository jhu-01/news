# 뉴스스탠드 디자인 및 시스템 가이드 (docs/design.md)

이 문서는 뉴스스탠드 웹 서비스의 UI/UX 구현을 위한 핵심 가이드라인입니다. 모든 컴포넌트 개발 및 스타일링은 이 문서의 스펙을 준수해야 합니다.

## 1. 프로젝트 개요
- **목적**: 데스크톱 퍼스트 한국어 뉴스 포털. 6x4 그리드 언론사 브라우징, 구독 관리, 언론사별 리스트 뷰 및 자동 롤링 뉴스 제공.
- **주요 기능**: 
  - 언론사 그리드 보기 (전체/구독)
  - 언론사 구독/해지 (Hover 시 노출)
  - 뉴스 티커 자동 롤링 (상단 2개 라인)
  - 리스트 뷰 (6초 주기 자동 페이지 전환 및 프로그레스 바)

## 2. 디자인 원칙 (Design Principles)
- **장식 자제 (Clarity)**: 그라데이션이나 그림자 지양. hairline 보더(#D2DAE0)로 구분.
- **타이포그래피 중심**: 언론사 로고는 이미지가 아닌 텍스트(Wordmark) 스타일로 구현.
- **포인트 컬러**: `#7890E7` (Indigo). 구독 배지, 활성 탭 프로그레스 바에만 제한적 사용.
- **밀도 있는 레이아웃**: 16px 본문, 12px 메타 데이터 사용. 여백을 통한 시각적 호흡 확보.

## 3. 컬러 토큰 (Color Tokens)
| Token | Value | Usage |
| :--- | :--- | :--- |
| `ink` | `#14212B` | 기본 텍스트, 강조 레이블, 진한 그리드 선 |
| `sub` | `#5F6E76` | 보조 텍스트 (날짜, 캡션, 구독 정보) |
| `mute` | `#879298` | 비활성 탭, 빈 셰브론 아이콘 |
| `line` | `#D2DAE0` | 1px 구분선, 카드 테두리, 그리드 스트로크 |
| `soft` | `#F5F7F9` | 티커 배경, 필드 탭 배경, 셀 Hover 배경 |
| `card` | `#FFFFFF` | 그리드 셀 배경, 리스트 뷰 본문 |
| `accent` | `#7890E7` | 구독 배지, 활성 탭 배경 |
| `accent-deep` | `#4362D0` | 활성 탭 내 프로그레스 바 채우기 |

## 4. 타이포그래피 (Typography)
- **기본 폰트**: Pretendard (Fallback: Noto Sans KR)
- **숫자 폰트**: IBM Plex Mono (탭 카운터 등)
- **세리프 폰트**: Noto Serif KR (일부 언론사 로고용)
- **주요 스펙**:
  - `display`: 24px/Bold (헤더 로고)
  - `heading`: 16px/Bold (활성 탭, 기사 제목)
  - `body`: 16px/Medium (비활성 탭, 날짜)
  - `list-item`: 14px/Medium (기사 리스트)
  - `caption/badge`: 12px/Medium (구독 버튼, 배지 숫자)

## 5. 레이아웃 스펙 (Spacing & Grid)
- **기본 단위**: 8px (4, 8, 12, 16, 24, 32, 48, 64 배수 사용)
- **전체 너비**: 1280px (중앙 컨텐츠 영역 930px)
- **그리드 구조**: 6열 x 4행 (총 24개 셀)
  - 셀 크기: 약 154px x 96px
  - 셀 간격: 1px (보더로 구현)

## 6. 핵심 컴포넌트 상세
### 6.1 뉴스 티커 (Ticker)
- 두 개의 라인이 독립적으로 3.2초마다 롤링.
- 0.55s 간의 크로스페이드(Cubic-bezier) 효과.
- 호버 시 롤링 일시 정지.

### 6.2 언론사 워드마크 (Press Wordmark)
- 언론사 로고를 텍스트 스타일로 동적 렌더링.
- Props: `color`, `bg`, `weight`, `family(sans/serif)`, `italic`, `underline`, `tracking`, `flag(아시아경제 등)` 등 포함.

### 6.3 리스트 뷰 & 프로그레스 (Opened Press)
- 상단 6개 카테고리 탭 존재.
- 현재 활성 탭은 6초 동안 왼쪽에서 오른쪽으로 `#4362D0` 색상으로 채워짐.
- 100% 도달 시 다음 기사 또는 다음 언론사로 자동 전환.

## 7. 상태 및 흐름 (States & Flows)
- **전체 언론사**: 3페이지 분량의 6x4 그리드.
- **내가 구독한 언론사**: 구독한 셀만 노출, 나머지는 빈 칸.
- **Hover**: 그리드 셀 호버 시 언론사 로고 대신 '구독하기' 또는 '해지하기' 버튼 노출.
- **Click**: 언론사 클릭 시 그리드 뷰에서 리스트 뷰로 전환.

## 8. 접근성 (Accessibility)
- Tablist/Tab 시맨틱 태그 준수.
- 셰브론 버튼에 `aria-label` 부여.
- 키보드 포커스(`:focus-within`) 시 호버와 동일한 구독 버튼 노출.
- `prefers-reduced-motion` 대응 (티커 및 프로그레스 애니메이션).
